# Carpeta de Imágenes de Productos

Esta carpeta contiene las imágenes de los productos de proteínas.

## Instrucciones

1. Guarda aquí las imágenes de los productos en formato JPG o PNG
2. Nombra las imágenes de forma descriptiva, por ejemplo:
   - `whey-protein-gold.jpg`
   - `proteina-caseina.jpg`
   - `proteina-vegana.jpg`
   - `whey-isolate.jpg`
   - `proteina-carne.jpg`
   - `mass-gainer.jpg`
   - `proteina-huevo.jpg`
   - `whey-concentrate.jpg`

3. Luego actualiza el archivo `inicializar.py` para que use las rutas locales en lugar de URLs de Unsplash

## Ejemplo de ruta en inicializar.py:
```python
"images": "/assets/productos/whey-protein-gold.jpg"
```

Una vez que subas las imágenes aquí, dime y actualizaré el código para usarlas.
